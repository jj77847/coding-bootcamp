## Difference between biggest 2 numbers

You have an array of integers. You need to calculate the difference between the 1st biggest number and the 2nd biggest number of the array.

```javascript
diffOfBiggestNumbers([10, 5, 2]);
```

In this case, the 1st biggest number is 10 and the 2nd biggest number is 5. So, the function returns 5, the result of 10 - 5.

## Difference between biggest and smallest numbers

You have an array of integers. You need to calculate the difference between the biggest number and the smallest number of the array.

```javascript
diffOfBiggestAndSmallest([10, 5, 2]);
```

In this case, the biggest number is 10 and the smallest number is 2. So, the function returns 8, the result of 10 - 2.
